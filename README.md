## Contents of repository
The [guide_images](https://github.com/BryanAnderson2019/se-bryan-app-deployment-guide/tree/main/guide_images) folder contains the images used in the guide.

The [Auto-Scaling-groups-guide.md](https://github.com/BryanAnderson2019/se-bryan-app-deployment-guide/blob/main/Auto-Scaling-groups-guide.md) file contains a guide on setting up an Auto Scaling group.

The [Cloudwatch-monitoring-alert-management-guide.md](https://github.com/BryanAnderson2019/se-bryan-app-deployment-guide/blob/main/Cloudwatch-monitoring-alert-management-guide.md) file contains a guide on setting up a monitoring dashboard as well as a CPU usage alarm.

The [Sparta-app-deployment-guide.md](https://github.com/BryanAnderson2019/se-bryan-app-deployment-guide/blob/main/Sparta-app-deployment-guide.md) file contains a guide on deploying the Sparta app.

The [creating-images-guide.md](https://github.com/BryanAnderson2019/se-bryan-app-deployment-guide/blob/main/creating-images-guide.md) file contains a guide on setting up an AMS image.

The [mongodb-deployment-guide.md](https://github.com/BryanAnderson2019/se-bryan-app-deployment-guide/blob/main/mongodb-deployment-guide.md) file contains a guide on deploying a MongoDB and having the Sparta app interact with it.

The [nodejs20-se-test-app-2025.zip](https://github.com/BryanAnderson2019/se-bryan-app-deployment-guide/blob/main/nodejs20-se-test-app-2025.zip) file contains the app and should be downloaded if you want to follow along with the Sparta app deployment guide.
